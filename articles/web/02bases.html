Introduction aux langages web

Notions principales

hypertext : document ou ensemble de documents contenant des unités d'informations liées entre elles par des hyperliens. Le langage permettant d'écrire de l'hypertext est le HTML.
HTML : l'Hyper Text Markup Langage est un langage de balisage permettant d'écrire et de structurer sémantiquement le contenu d’une page web. Sur une page web, on écrit les éléments entre deux balises html, une ouvrante et une fermante (on écrira un / avant le nom de la balise pour la fermer). < balise > Ceci est un titre < /balise > (sauf cas particulier des balises images et br)
CSS : Cascading Style Sheets ou feuille de style en cascade est un langage pour le web permettant la mise en forme des balises HTML. Ce langage permet de gérer l'agencement, le positionnement et la décoration du contenu d'une page web.
Le code CSS fait donc référence à des balises écrites dans notre page html et s'écrit :
balise {
propriété1:valeur;
propriété2:valeur;
}
On parle actuellement d'HTML 5 et CSS 3 car les langages ont évolué depuis leur création en ajoutant certaines propriétés et noms de balises.

⚠ Ces deux langages ne tolèrent aucune faute de syntaxe

Aucun logiciel spécifique n'est nécessaire pour écrire ces deux langages, il est possible d'écrire en html et css dans un simple éditeur de texte. Ici, le logiciel utilisé est Sublime text, un logiciel gratuit et très pratique dans son utilisation. Pour information la suite Adobe a développé, dès 1997, son propre éditeur de texte pour le web nommé Dreamweaver, c'est un logiciel WYSIWYG (what you see is what you get) très peu ulitisé par la communauté des développeurs car créant beaucoup trop de balises inutiles dans le code.

Le langage html

La première page d'un site web se nomme obligatoirement index.html car le navigateur web ira toujours chercher ce nom de fichier en premier à l'url indiquée. Pour rappel l'url est la localisation d'un document hypertextuel (ou pour le web, de notre serveur).

La page html se constitue de deux parties, la partie < head > le “cerveau” de la page dans lequel vont se situer toutes les informations techniques caractérisant la page (la langue, l'encodage des caractères, les mots clés, le titre...) Et le < body > partie ou l’on aura tout notre contenu (texte, image, titre, bloc...)

Écriture des titres

Il existe en html 6 niveaux de titres, du plus au moins important. Ces titres permettent d'indiquer au navigateur et surtout aux robots de référencement ce qui est important dans la page web. Généralement on utilisera qu'un seul titre de niveau 1 ce qui voudra dire que c'est l'élément le plus important de la page.

Écriture des paragraphes

Un paragraphe en html permet d'écrire du texte courant dans une page. On peut utiliser autant de fois que l'on veut la balise paragraphe dans une page. Il faudra juste bien penser à la refermer à chaque fois qu'on l'a ouverte. Une des erreurs les plus fréquentes lorsque l'on commence à coder est l'oubli de fermeture de balises.

Mise en valeur d'un mot dans un paragraphe

Pour le référencement et que le navigateur web comprenne qu'un mot ou groupe de mot est important sémantiquement dans un paragraphe, il existe deux balises permettant de mettre en valeur des mots ; la balise < strong > (la plus importante) et la balise < em > .

Les listes

Il existe deux types de listes en html, la liste ordonnée et la liste non-ordonnée. Comme son nom l'indique la première donnera un ordre à chaque élément selon l'ordre dans lequel les éléments seront écrits. Alors que pour la deuxième chaque élément de liste sera au même niveau. Les listes sont généralement utilisées en html pour écrire les menus.

Les liens

vers une autre page de son site

La balise html permettant de faire des liens est la balise < a > à laquelle on ajoutera l'attribut href="" pour indiquer vers ou mène le lien. Le premier type de lien est celui qui permet de lier deux pages entre elles dans un site, on fera un lien d'un fichier vers un autre fichier.

vers un autre site web

Il est aussi possible de faire des liens externes à son site en indiquant l'URL à atteindre à la place du nom du fichier. Dans ce cas on utilisera l'attribut target="_blank" pour indiquer au navigateur d'ouvrir le lien dans un nouvel onglet et ainsi éviter de perdre l'utilisateur.

lien de téléchargement pdf

Pour faire un lien de téléchargement pdf, il suffit de lier un mot, ou groupe de mot, vers un fichier pdf généralement ranger dans un sous-dossier data sur son serveur.

lien vers un mail

Pour faire un lien vers une boite mail on utilisera aussi la balise < a > avec son attribut href="" dans lequel on ajoutera la propriété mailto:

lien vers une ancre

Les liens vers des ancres sont utilisés dans le cas des sites one-page. Il est possible de faire non pas un lien vers une autre page de son site mais vers une zone de sa page. Dans ce cas, il y a deux choses indispensables à faire. Dans un premier temps on va identifier la zone vers laquelle va renvoyer notre lien. Il nous suffira pour ce faire d'ajouter un id="nomAncre" à notre balise ciblée.

Dans un second temps il suffira d'indiquer href="#nomAncre" dans la balise a de l'élément qui renverra vers l'ancre.

Les images

Avant d’insérer une image dans sa page il faut bien la ranger dans le dossier data à la racine de son dossier www. Il ne faudra mettre aucun espace ni aucun accent au nom du fichier image. On utilise la balise < img > à laquelle on ajoute l'attribut src="" pour indiquer la source de l'image (où elle est rangée sur notre serveur). Pour le web on utilise le format jpg pour les images sans transparence, png pour celles avec de la transparence et gif pour les images animées. On pourra aussi utiliser le svg pour reduire leurs poids. En effet, le poids des images est une contrainte très importante lors de l'intégration web, on voudra toujours gagner en temps d'affichage des pages et plus les images seront lourdes plus la page mettra du temps à se charger.

Le langage CSS

nomenclature et fonctionnement

Il est possible d’attribuer du style à des balises HTML de trois manières, soit directement à l’intérieur d’une balise, soit en écrivant le code CSS dans la partie < head > de la page, ou bien la manière la plus propre, écrire les styles dans un fichier à part que l’on liera avec la page html en écrivant cette ligne de code dans la partie < head > de la page :
< link rel="stylesheet" href="style.css" >
Grace à cette ligne de code on indique au navigateur que le style qui sera attribué aux balises html de la page sera situé dans un fichier nommé style.css rangé dans le même dossier que le fichier html.

couleur

La couleur en css peut s'écrire de trois façons différentes. Elle peut s'écrire par sa valeur hexadécimale par exemple #ffffff pour le blanc, par sa valeur en rgb(255,255,255) soit directement par son nom par exemple white. Pour modifier la couleur de fond d'une page on utilisera la propriété background-color que l'on appliquera à la balise body

le reset d'Eric Meyer

Avant d'aller plus loin dans les propriétés CSS il faut savoir que chaque navigateur applique des propriétés aux balises des pages qu'ils affichent, par exemple Google Chrome pourrait appliquer une certaine valeur de marge à chaque balise article alors que Firefox en appliquerait une autre valeur. De ce fait il est important de remettre à zéro toutes les propriétés CSS que pourraient appliquer les navigateurs web avant de commencer à mettre en forme ses balises html. On utilise pour cela les lignes de codes CSS écrites par Eric Meyer que l'on vient positionner dans un fichier reset.css.
